# Pythonを用いたシューティングゲーム開発
60秒間の間に君は多くの敵を倒せるか...

# 操作方法
* スペースキーで球を発射
* ↑で機体を前進
* ↓で機体を後退
* ←で機体を左へ移動
* →で機体を右へ移動

# ルール
* 制限時間は60秒
* 制限時間のこり60秒から11秒は常に敵が3体出現
* 制限時間のこり10秒から0秒は常に敵が5体出現
* 60秒間敵を倒しつつも生き残ればClear
* 敵に当たったり、球を受けるとGameOver
